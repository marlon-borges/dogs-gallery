# PT-BR | Dogs gallery

Básica visão geral sobre esse projeto de estudo

## Descrição

Estou em um sistema de estudo de front end, criando alguns simples e pequenos projetos de HTML, CSS e JavaScript.

Este projeto foi criado com uma API gratuita de [dog.ceo](https://dog.ceo/api/breeds/image/random) com um componente para gerar a interação de galeria com [fancybox](https://fancyapps.com/fancybox/).

# EN | Dogs gallery

Basic overview of this study project

## Description

I am in a front-end study system, creating some simple and small HTML, CSS, and JavaScript projects.

This project was created with a free API from [dog.ceo](https://dog.ceo/api/breeds/image/random) with a component to generate the gallery interaction with [fancybox](https://fancyapps.com/fancybox/).
